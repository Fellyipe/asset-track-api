# AssetTrack API

API Rest para gestão de ativos e manutenção operacional, permitindo o controle de ordens de serviço, inspeções e relatórios técnicos.


## 🛠️ Tecnologias
- Java 17
- Spring Boot 3
- Spring Security (JWT)
- JPA/Hibernate
- PostgreSQL
- Docker

## 🚀 Como rodar

1. **Clone o projeto:**
   ```bash
   git clone https://github.com/Fellyipe/asset-track-api
   cd asset-track-api
   ```

2. **Suba o container (Docker):**
   ```bash
   docker-compose up -d
   ```

3. **Inicie a aplicação:**
   ```bash
   ./mvnw spring-boot:run
   ```
